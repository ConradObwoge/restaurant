# Maakaso Kitchen - Online Ordering System

## Setup Instructions

### 1. Hosting Setup
- Buy domain + hosting (Truehost, HostPinnacle, or 000webhost for free)
- Upload all files via cPanel File Manager or FTP

### 2. Configuration
- Edit `config.json` with your phone number and M-Pesa Till number
- Change admin password in `admin/login.html` (default: admin123)

### 3. Delivery Zones
- Edit `delivery/zones.json` to add/remove delivery areas
- Highlight areas where Glovo does NOT reach!

### 4. Testing
- Open `index.html` in browser
- Add items to cart, fill details, place order
- Check `admin/dashboard.html` to see orders (login: admin123)

### 5. Sharing with customers
- Print QR code that points to your website URL
- Share link on WhatsApp groups in Kyumbi, Mua Hills, Kola, Joska

### 6. Marketing Message
"We deliver where Glovo doesn't reach! Order directly from us - no commission fees."
